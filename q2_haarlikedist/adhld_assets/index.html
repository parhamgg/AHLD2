RF Gram Matrix

This plot shows the RF affinity between each set of points. The RF affinity is a distance metric that measures how similar two samples are by comparing how similarly the decision trees in a random forest behave when classifying samples. In simpler terms this can be viewed as a distance matrix for all samples based on the RF affinity distance.

Reconstructed haar-like distance matrix

Our goal is to reconstruct the RF Gram Matrix using a reduced feature space, so we learn weights for the 5 most important internal nodes of the phylogeny tree and reconstruct the dataset using only this information. Ideally the reconstructed matrix will look similar to the RF Gram Matrix and have a relatively high correlation. Note that because the two different ways of reconstructing the data matrix use different scales, correlations in the range .2 - .5 should be considered adequate.
